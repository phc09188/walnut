## 주제 : 온라인 쇼핑몰(이름:호두)
### 목적 : 온라인 쇼핑몰을 만들어보면서 쇼핑몰의 구조를 파악해보자.

## 기술 스택
- [ ] Front : Thymleaf Html5 CSS JS
- [ ] Back : Springboot SpringSecurity 
- [ ] DB : Mysql Redis
- [ ] Mail : Google SMTP
